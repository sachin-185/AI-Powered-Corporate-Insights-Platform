## 📖 Overview

**SmartCorp AI** is an advanced, full-stack analytical dashboard built to empower HR and enterprise management teams. Instead of relying on static spreadsheets, this platform dynamically evaluates employee sentiment, forecasts critical KPIs, and flags high-risk departments before issues escalate. 

By leveraging native GPU-accelerated Natural Language Processing (NLP) alongside classic Machine Learning, SmartCorp uncovers the hidden stories within your company's data.

## ✨ Key Capabilities

*   **📊 Predictive Risk Modeling:** Utilizes a custom-trained `DecisionTreeClassifier` to label departmental risk (Low/Medium/High) based on cascading thresholds in attrition, engagement, and project delays.
*   **🗣️ Contextual Sentiment Analysis:** Feeds raw employee feedback into a fine-tuned `DistilBERT` model to automatically categorize company morale into Positive, Neutral, or Negative bands.
*   **📈 Automated Forecasting:** Enhances standard Altair time-series visualisations with mapped `LinearRegression` forecast lines, projecting future departmental performance.
*   **⚡ Real-Time Interactive UI:** A zero-latency Streamlit frontend allowing managers to aggressively filter metrics by date, KPI, and specific departments.
*   **📝 NLP Meeting Summarization:** Pre-configured Hugging Face `BART` pipeline to condense massive departmental transcripts into digestible executive summaries.

## 🏗️ System Architecture & Tech Stack

The platform is cleanly separated into two main pipelines: **Data Processing (`app.py`)** and **Visualization (`visual.py`)**.

| Category | Technology | Purpose |
| :--- | :--- | :--- |
| **Frontend UI** | Streamlit, Altair | Reactive dashboard, interactive charting, and user controls. |
| **Deep Learning** | PyTorch, Hugging Face | Local, GPU-enabled inference for zero-shot text classification and summarization. |
| **Machine Learning** | Scikit-Learn | Training local decision trees for risk assessment and regression lines for forecasting. |
| **Data Engine** | Pandas, NumPy | In-memory massive dataset manipulation, aggregation, and timezone handling. |

## 🚦 Getting Started

### Prerequisites
*   Python 3.9 or higher
*   *(Optional but recommended)* CUDA-compatible GPU for accelerated PyTorch execution.

### Local Installation

**1. Clone the repository**
```bash
git clone https://github.com/your-username/smartcorp-ai.git
cd smartcorp-ai
```

**2. Isolate your environment**
It's highly recommended to use a virtual environment to prevent package conflicts.
```bash
python -m venv env

# On Windows:
env\Scripts\activate
# On macOS/Linux:
source env/bin/activate
```

**3. Install Core Dependencies**
```bash
pip install streamlit pandas numpy scikit-learn altair transformers torch
```

**4. Boot the Dashboard**
Initialize the application. On the first run, Hugging Face will briefly download the required NLP model weights.
```bash
streamlit run visual.py
```
> 🌐 **Access the UI:** The terminal will output a local network address, typically `http://localhost:8501`.

---

## 🖥️ Navigating the Dashboard

1.  **Sidebar Filtering:** Start by isolating your target data. Select specific departments (e.g., HR, Sales) and dial in the date range. 
2.  **Risk Audit:** Check the top table generated by the AI classifier to immediately spot departments flagged as **"High"** risk.
3.  **Sentiment Mapping:** Scroll down to the stacked bar chart to see if negative sentiment correlates with higher risk flags.
4.  **Export CSV:** Once you have filtered down to a specific anomaly (e.g., Q3 Engineering delays), use the sidebar download button to pull the raw CSV for external presentation.
